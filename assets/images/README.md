# Logo Image Instructions

## Required Image

Please save the Pragati Engineering College logo as:
- **Filename:** `pragati-logo.png`
- **Location:** `assets/images/pragati-logo.png`
- **Recommended Size:** 500x500 pixels or larger (square format)
- **Format:** PNG with transparent background (preferred) or white background

## How to Add the Logo

1. Save the college logo image from your source
2. Rename it to `pragati-logo.png`
3. Place it in the `assets/images/` folder
4. The logo will automatically appear in:
   - Sidebar header (main dashboard)
   - Login page header
   - Any other branded sections

## Logo Specifications

- **Sidebar Logo:** Displayed at 50x50 pixels with circular background
- **Login Logo:** Displayed at 100x100 pixels with circular background
- **Format:** The logo will be automatically styled with proper padding and shadows

## Current Status

The HTML and CSS are already configured to use the logo. You just need to add the image file.
